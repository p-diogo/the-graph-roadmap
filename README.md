# The Graph Core Dev Roadmap

Core development strategic roadmap for The Graph protocol (2025-2026).

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Development

### Prerequisites

- Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)
- Bun (recommended) or npm

### Getting Started

```bash
# Clone the repository
git clone https://github.com/p-diogo/the-graph-roadmap.git
cd the-graph-roadmap

# Install dependencies
bun install
# or: npm install

# Start development server
bun run dev
# or: npm run dev

# Visit http://localhost:8080
```

### Building for Production

```bash
bun run build
# or: npm run build
```

### Running Tests

```bash
bun run test
# or: npm run test

# Watch mode
bun run test:watch
# or: npm run test:watch
```
